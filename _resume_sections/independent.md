### Agentia | TypeScript, React, Ink
A command-line interface (CLI) engine for orchestrating multi-agent workflows using locally hosted Large Language Models (LLMs).
- Developed an interactive CLI application utilizing TypeScript, React and Ink.
- Engineered for local execution and seamless integration with locally hosted infrastructure (llama.cpp hosted LLMs and Docker containers) with aim to build a modern, secure, privacy-first LLM tool.
